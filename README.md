# Medical Login Database

A simple C program that simulates a medical system login with a basic database.  
The program allows users to log in using stored credentials and demonstrates basic concepts such as user authentication, input handling, and simple data storage.

This project was created as a learning exercise for working with C programming, user input, and simple database-like structures.

## Features

- User login system
- Basic username and password verification
- Simple credential storage
- Command-line interface
- Demonstrates input handling in C

## Technologies Used

- C Programming Language
- Standard C libraries
- Command-line execution

## How to Run

1. Clone the repository:

git clone https://github.com/Canadian599xx/medical-login-database.git

2. Navigate into the project folder:

cd medical-login-database

3. Compile the program:

gcc main.c -o medical-login

4. Run the program:

./medical-login
