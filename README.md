# Medical Login Database

A simple Java program that simulates a medical system login with a basic database. The program allows users to log in using stored credentials and demonstrates core programming concepts such as user authentication, input handling, and simple data storage.

This project was created as a learning exercise for working with Java, user input, and basic login system logic.

## Features

- User login system
- Username and password verification
- Simple credential storage
- Command-line interface
- Demonstrates input handling in Java

## Technologies Used

- Java
- Standard Java libraries
- Command-line execution

## How to Run

1. Clone the repository:

git clone https://github.com/Canadian599xx/medical-login-database.git

2. Navigate into the project folder:

cd medical-login-database

3. Compile the program:

javac Main.java

4. Run the program:

java Main

This project is intended for educational purposes to practice:
